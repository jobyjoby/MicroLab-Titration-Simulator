#  Micro Lab Simulator: Guided Titration

This project is a Unity-based simulation of a titration workflow designed as part of a technical assessment.

##  Features

- Step-by-step guided UI workflow
- Enforced order using Next button + action validation
- Burette to flask volume simulation
- Interactive stopcock (slider-based flow control)
- Endpoint detection with color change
- Initial and final readings with volume calculation

##  Workflow

1. Start Simulation
2. Fill Burette
3. Set Initial Reading
4. Add Indicator
5. Perform Titration (slider control)
6. Endpoint detection
7. Result display

##  Tools Used

- Unity (3D Core)
- C#
- Unity UI (UGUI)

##  How to Run

1. Open project in Unity
2. Load `MainScene`
3. Press Play
4. Follow on-screen instructions
